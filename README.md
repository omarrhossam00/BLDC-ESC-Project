# BLDC Motor Controller

A BLDC motor controller using gate driver and MOSFET inverter.

## Features
- Three-phase inverter
- PWM control
- Bootstrap circuit
- Simulink simulation
- Hardware implementation

## Tools
- MATLAB Simulink
- IR2110
- MOSFETs
- Power Electronics
- Proteus
